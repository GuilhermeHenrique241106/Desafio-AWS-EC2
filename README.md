Desafio de Projeto: Gerenciamento de Instâncias na AWS

Este repositório contém a entrega do desafio de computação em nuvem da DIO. O objetivo foi compreender os principais conceitos de infraestrutura como serviço (IaaS) utilizando a AWS.

Conceitos Aprendidos

Amazon EC2: Funcionou como o servidor/computador virtual que criamos na nuvem para rodar nossas aplicações.
Amazon EBS:** Atua como o "HD" ou unidade de armazenamento anexada ao nosso servidor EC2 para salvar os dados.
AMI (Amazon Machine Image):** O modelo pronto (sistema operacional) que escolhemos para iniciar o servidor rapidamente.
Par de Chaves (Key Pair):** O mecanismo de segurança essencial que geramos para garantir o acesso restrito e seguro à instância.
